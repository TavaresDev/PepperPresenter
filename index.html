Pepper, a social robot

Pepper, the social robot

Pepper is a 120 cm tall humanoid robot capable of interacting with humans in an autonomous way.

The robot is social, meaning that it is able to understand and express emotions and personalities, acting in a consequent manner according to the situation.

The application, Pepper Presenter App

The image above is a quick introduction on how the application works.

Written in Python, and running from any machine with Python 2.7 and the necessary dependencies available, it controls remotely the robot, making it present the PowerPoint: Pepper will tell the story found in the presentation notes while gesturing and expressing emotions.

As extra functionalities, the robot is also gifted with extra behaviours, triggered by the user by adding in the presentation notes [bracketed] instructions or emoticons. Thanks to that, Pepper will also be able to show emotions during a sentence or point in a determined zone of the screen to attract the attention there.

To perform those extra functionalities, a PowerPoint add-in is included with the app, letting the designer export the presentation in a way the Presenter Application can understand.

In detail

Requierements:

A LAN where both the robot and the computer are connected is necessary

    Presenter app

  • Python 2.7
  • naoqi Python SDK
  • Python modules: python-pptx, lxml, htmlPy, PySide

    Exporter app

  • PowerPoint 2007 and above

User defined behaviours:

    Emotions

    Adding those icons in the text, the robot will express such emotion
  • :) - happiness
  • :( - sadness
  • :| - indiference

    Pointing

    Adding this bracketed pseudo-code, the robot will point to a determined screen zone (x = 1 to 4)
  • [point=x]
