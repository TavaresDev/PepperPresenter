PepperPresenter App Help

To make Pepper present your keynote these steps should be followed:

  1. Know your robot IP direction and type it in the format xxx.xxx.xxx.xxx in the Robot IP field
  2. Use the file picker to select the already exported [using the PowerPoint extension] .pptx file
  3. Click on Start presentation and enjoy. Everything should go straight forward
  4. After Pepper has finished the presentation click on the image to go back to the app

About

This software is written in Python. The GUI has been created in HTML5 and Bootstrap3 CSS

For information about the usage procedure refer to Help section

Basic requirements to run this software:

  • Python 2.7
  • NAOqi SDK for Python
  • External Python modules: PythonPPTX, htmlPy, lxml, PyQt4, PySide

An exported .pptx file and dependencies obtained using the PepperPresenter PowerPoint Exporter extension is also needed. This PowerPoint extension can be found on Github [see below]
